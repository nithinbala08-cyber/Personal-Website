# Personal Website

A modern, responsive personal website with a clean and minimalist design.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: Elegant fade-in and scroll animations
- **Interactive Navigation**: Smooth scrolling navigation with active section highlighting
- **Typing Animation**: Dynamic typing effect for profession display
- **Modern UI**: Clean, minimalist design with beautiful gradients and shadows
- **Mobile Menu**: Hamburger menu for mobile devices

## Sections

1. **Hero Section**: Introduction with name, profession, and call-to-action buttons
2. **About Section**: Personal information and skills showcase
3. **Projects Section**: Portfolio of featured projects
4. **Contact Section**: Social media links and contact information

## Setup

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content with your personal information

## Customization

### Personal Information

1. **Name**: Replace "Your Name" throughout the HTML file
2. **Profession**: Update the typing animation professions in `script.js` (line 6)
3. **About Text**: Edit the about section content in `index.html`
4. **Skills**: Modify the skills tags in the about section
5. **Projects**: Update project cards with your actual projects
6. **Contact Links**: Replace placeholder URLs with your actual social media profiles

### Styling

- **Colors**: Modify CSS variables in `styles.css` (lines 4-11)
- **Fonts**: Change the Google Fonts import in `index.html` (line 10)
- **Images**: Replace SVG placeholders with your actual images

### Adding Your Photo

Replace the SVG placeholder in the hero section with your actual photo:

```html
<div class="hero-image">
    <img src="path/to/your/photo.jpg" alt="Your Name" />
</div>
```
